Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>(não semântica).

Nesta frase temos um termo em destaque usando a tag <strong>(Contêm semantica, tem um sentido).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i>(Não semântica).

Nesta frase temos um termo em destaque usando a tag <em>(Contêm semantica, tem um sentido)

Texto marcado:

Nesta frase temos um termo marcado usando a tag <mark>, aqui não iremos mudar a cor, pois é uma tag sem

Texto Grande e Pequeno

Aqui tem um texto grande! usando a tag <big>.

Aqui tem um texto pequeno! usando a tag <small>.

Texto deletado

Podemos marcar um texto que foi excluído, que deve ser lido mas não considerado, usando a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usamos a tag <ins>.

Também existe a tag <u>, porém, ela não possui significado semântico.

Texto sobrescrito

Usamos a tag <sup> para sobrescrever um termo.

Texto subescrito

Usamos a tag <sub> para subrescrever um termo.
